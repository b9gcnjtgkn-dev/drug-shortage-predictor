⚙️ Solution & Key Features

Not a concept.
A live, working product.

📡

Live Data Pipeline

Auto-connects to ClinicalTrials.gov API v2. Fetches & normalizes 200+ real terminated trial records in real-time.

Python · Requests · Pandas · CSV Export
🧠

NLP + AI Classifier

TF-IDF vectorizer converts termination text to features. Random Forest predicts shortage risk at 78% accuracy.

scikit-learn · TF-IDF · Random Forest · .pkl export
⚡

High-Speed Prediction API

FastAPI backend loads model in-memory. POST any termination text → receive risk score + HIGH/LOW flag instantly.

FastAPI · Uvicorn · Python 3.11 · Swagger /docs
🖥️

Explainable AI Dashboard

Next.js UI shows risk probability, source text, and historical data. No black box. Every alert is traceable.

Next.js · React · Glassmorphism CSS · localhost:3000
